IssueHub

The client for GitHub on iOS and Android

About

IssueHub is the unofficial client for GitHub Issue on the iOS and Android platform. The project is designed in C# using the Xamarin product with the expectation that it will one day operate on all major mobile platforms. IssueHub is not, in any way, affiliated with GitHub, Inc.
